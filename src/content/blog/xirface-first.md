### A new .NET Game Development Framework
Xirface is a new Vulkan-based lightweight .NET game development framework built on Silk.NET, featuring mesh-based rendering, GLSL shader support, mesh-based custom text rendering, a JSON driven UI system and a simple asset loading pipeline supporting images and .spv (compiled GLSL) shaders.

### Version 0.1.0
Version 0.1.0 is the first public release of the framework, it only features a limited subset of features, mesh-based rendering, asset loading and input detection, which can be interacted with by the game loop.
### Future
The framework is still on rapid development, and most mentioned features are close to completion and are expected to be released in the following weeks.
